1 Chapter Exercises

  1. length :: [a] -> Int

    length takes a list, of any type, and returns an int, representing the number of elements in the list.

    1. 5
    2. 3
    3. 2
    4. 5

  3. The latter will error due to length returning an int instead of a Fractional

  4. Prelude> 6 `div` length [1, 2, 3]

  5. The type will be a bool, we’d expect a True result.

  6. The type again will be a bool, this time False.

    1. This will work, and will return True.
    2. This won’t work as lists must be homogenous.
    3. This will return an int.
    4. This will return False since ‘b’ is not less than ‘a’.
    5. This will error as Haskell does not have truthy vales.
  8. isPalindrome :: (Eq a) => [a] -> Bool
isPalindrome x = x == reverse x
  9. myAbs :: Integer -> Integer
myAbs x = if x < 0 then -x else x
  10. f :: (a, b) -> (c, d) -> ((b, d), (a, c))
f t1 t2 = ((snd t1, snd t2), (fst t1, fst t2))
f t1 t2 = map (snd, fst) (t1, t2)
