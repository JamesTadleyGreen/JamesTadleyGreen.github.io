1 Editor

I use NeoVim. This has mssivly helped my productivity. Prior to this I used VSCode, the plugin system VSCode has is incredible, and obviously very user friendly. Currently, I haven’t utilised NeoVim’s lua library to the extent I would like.

2 Terminal

I use the Windows Terminal this is shockingly good. At some point I might move away from this. I also use TMux for my terminal sessions. Combining this with tmux-resurrect and tmux-continuum allows me to keep my terminal instanes between restarts of my PC. This is such a nice workflow that emulates some of VSCode’s auto-opening of tabs.

3 Auto-complete

I use Codeium, simply because it is a great free tool, and I don’t want to pay for Copilot.

4 OS

I use WSL, specifically an Ubuntu installation. I would like to be on a native [Linux] installation, but unfortunately I have to interact with Microsoft Office.

5 Source Control

I use Git, and GitHub.
